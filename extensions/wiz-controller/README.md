# Wiz Controller

Control your Wiz lights directly from Raycast

## Setup


You will require finding the ip address of the light. To do that, check this out:
  1) Open the WiZ app and go to the room containing the light.
  2) Long-press the light icon or tap the light to open its settings.
  3) Tap the dropdown arrow next to the light's name at the top.
  4) Select "Device Info".
  5) The IP address will be displayed.
