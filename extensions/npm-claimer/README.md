# npm Claimer

Claim npm package names instantly, right from Raycast.

## Features

- **One-click claiming** — Just type the package name and hit enter
- **Instant availability check** — Know immediately if the name is taken
- **Zero config** — No project setup, no terminal, no hassle

## Setup

1. Get your npm access token from [npmjs.com](https://www.npmjs.com/settings/~/tokens)
2. Paste it in the extension preferences
3. Start claiming!

## Usage

1. Open Raycast and search for "Claim npm Package"
2. Enter your desired package name
3. Done — your package name is reserved
