# FitDesk Changelog

## [1.0.0] - 2026-02-03

### Added

- Initial release
- 24 exercises across 5 categories (upper body, core, lower body, cardio, full-body)
- Start Workout command for random exercise selection
- Browse Exercises command with category filtering
- View Statistics command with daily, weekly, and monthly tracking
- Exercise Reminder menu bar command with configurable intervals
- GIF animations for all exercises
- 10-second preparation countdown before exercises
- Audio notifications for exercise start and completion
- Exercise history tracking
- Streak tracking for consecutive days of exercise
