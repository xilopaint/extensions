# Cursor Costs Changelog

## [Initial Release] - 2026-01-30

- Add menu bar command showing Cursor API usage and costs in real-time
- Add list view with detailed model breakdown and statistics
- Add configurable menu bar title display (percent, cost, usage, or both)
- Add progress circle icon option based on usage percentage
- Add date range selection (billing cycle or calendar month)
- Add customizable section visibility (subscription, cost, models, tokens)
- Add model info display options (cost, tokens, or both)
- Add token format selection (short or full numbers)
- Add optional refresh and settings buttons in menu
- Add subscription info display with usage limits and bonus tracking
- Add free usage tracking for free models
- Add token statistics breakdown (input, output, cache read/write)
- Add configurable refresh interval (1min to 30min)
- Add cost format customization (decimal places, dollar sign)
- Add quick actions: refresh, open dashboard, copy values
- Add secure token storage using Raycast encrypted preferences
- Add support for both light and dark themes
