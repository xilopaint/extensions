# PDSls

[AT Protocol](https://atproto.com) utilities directly from Raycast. This extension integrates with [pdsls.dev](https://pdsls.dev) to help developers inspect data on the network.

## Features

- Search for profiles and quickly open them in PDSls.
- Resolve identities bidirectionally.
- Inspect the complete operation log for `did:plc` identities.
- View lexicon schema documentation rendered in Markdown.
