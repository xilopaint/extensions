# GitCDN Changelog

## [Initial Version] - 2026-01-30

### Added

- Browse files from GitHub repositories
- Support for multiple GitHub URL formats (owner/repo, full URLs, branch-specific)
- Automatic recursive search for files in subdirectories
- CDN URL generation using jsDelivr
- Raw GitHub URL generation
- Image preview with details view
- Copy CDN and raw URLs to clipboard
- Open files in browser
- Search and filter files by name
- Support for common image formats (PNG, JPG, GIF, WebP, SVG, ICO, BMP)
