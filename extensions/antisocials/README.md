# AntiSocials

Open social media links via privacy-friendly alternative frontends. Just copy a link to your clipboard and run the command - it'll open in your browser without the tracking.

## Supported Platforms

| Platform    | Frontend                           |
| ----------- | ---------------------------------- |
| X / Twitter | [xcancel.com](https://xcancel.com) |
| Instagram   | [Imginn](https://imginn.com)       |

## Usage

1. Copy an X/Twitter or Instagram URL to your clipboard
2. Run "Open Link Antisocially" from Raycast
3. The link opens in your default browser via the privacy frontend

## Acknowledgments

This extension is only possible thanks to the developers and maintainers of these privacy-respecting frontends:

- **[xcancel.com](https://xcancel.com)** - Alternative Twitter/X frontend (based on Nitter)
- **[Imginn](https://imginn.com)** - Instagram viewer without tracking

These projects enable browsing social media without invasive tracking, ads, or requiring an account.
