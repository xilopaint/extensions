# 42 API Changelog

## [Initial Version] - 2026-01-26

### Features
- Find User: Search for 42 users and view their current cluster location
- Check Logtime: View location statistics for any user over the past 30 days
- Today's Logtime: Menu bar extra showing your daily logtime with goal tracking
- Confetti celebration when daily logtime goal is reached
- OAuth authentication with automatic token refresh
- Debug mode for troubleshooting API issues
