# YouTube Highlights Changelog

## [Initial Version] - 2026-01-27

- Create highlights from YouTube videos playing in Arc browser
- Capture configurable duration before current timestamp
- Add notes to highlights
- View all highlights in chronological order
- Search highlights grouped by video
- Full-text search across highlights and video titles
- Export highlights to Markdown, JSON, or CSV
- Sign in for cloud backup
- Sync highlights across devices (Pro feature)
