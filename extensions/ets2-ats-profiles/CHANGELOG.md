# ETS2-ATS Profiles Changelog

## [Initial Version] - 2026-01-26

- Added commands to manage Euro Truck Simulator 2 profiles
- Added commands to manage American Truck Simulator profiles
- Profile viewing with detailed information:
  - Company name
  - Experience points
  - Distance traveled
  - Customization points
  - Current truck brand
  - Active mods list
  - Creation and last saved timestamps
- Profile management features:
  - Rename profiles
  - Duplicate profiles
  - Delete profiles with confirmation
  - Show profile folder in Finder
- Sorting options:
  - Sort by name (A-Z)
  - Sort by last saved time
  - Sort by creation time
- Copy actions:
  - Copy parsed profile data as JSON
  - Copy profile name
  - Copy profile path
- SII file parser command for advanced users
- Automatic SII file decryption using TypeScript library
- Windows platform support
