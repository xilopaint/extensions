# TorBox Changelog

## [Added multi-type downloads, file view, and more] - 2026-01-28

- Add support for web downloads and usenet
- Add support for queued downloads
- Add download status indicator (Ready / progress percentage / Queued)
- Add "Refresh All Downloads" action (Cmd+R)
- Add "Delete Download" action (Ctrl+X)
- Add "View Files" action for multi-file downloads (Cmd+Enter)
- Show file count for downloads with multiple files
- Show all downloads instead of last 10
