# Botpress

Manage your [Botpress](https://botpress.com) chatbots directly from Raycast.

## Features

- **List Bots** - View all chatbots in your Botpress workspace
- **Open in Studio** - Jump straight into the Botpress Studio to edit your bot
- **Open in Dashboard** - Access your bot's dashboard settings
- **Copy Bot ID** - Quickly copy a bot's ID to clipboard

## Setup

1. Get your **Personal Access Token** from [Botpress Cloud](https://app.botpress.cloud):
   - Click your profile icon (top right)
   - Go to **Account Settings**
   - Select the **Access Tokens** tab
   - Click **Generate new token** and copy it

2. Get your **Workspace ID** from the URL when logged into Botpress Cloud:
   - `https://app.botpress.cloud/workspaces/{workspaceId}/...`

3. Enter both values in the extension preferences when prompted

## Commands

| Command | Description |
|---------|-------------|
| List Bots | View and manage all bots in your workspace |
