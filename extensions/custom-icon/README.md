# Custom Icon

## Features
- Customize apps, folders or files icon
- Use an image or an emoji, add a custom background (color, gradient, image or default folder icon)

## Options
- `Icon Image` - Path to the image to use
- `Pick Emoji` - Emoji to use
- `Add Background` - Add a background with rounded corners
- `Background Type` - Color, Gradient or image
- `Restart App` - Restart the app after changing the icon (if running)
- `Use Folder Icon` - Overlay on default macOS (Tahoe) folder icon

### Adjust padding

Adjust padding of custom background in preview using Actions:
- Increase padding (+5%) `⇧` `↑`
- Decrease padding (-5%) `⇧` `↓`
- Fine increase (+1%) `⌥` `⇧` `↑`
- Fine decrease (-1%) `⌥` `⇧` `↓`

## Emoji Source
The emojis used in this extension are sourced from the [emoji-data](https://github.com/iamcal/emoji-data) project.
